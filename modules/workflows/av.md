# OmniPM 音视频型项目工作流

> 模块名称：av_workflow | 版本：0.2.0 | 依赖：modules/roles.md(category=av), modules/design-dimensions.md

## 概述
音视频型项目以内容制作流程为核心，强化脚本策划、分镜设计、制作排期和后期制作管理能力。适用于教程视频、播客制作、宣传片、直播策划等项目类型。

## Step A 差异化：音视频制作设计

在通用 D1-D7 中，音视频型项目强化以下 5 个维度：

### D-AV1. 内容策划
- 节目/视频定位与风格定义（教育/娱乐/纪实/商业/混合）
- 目标受众与观看/收听场景分析
- 内容类型选择（教程/访谈/Vlog/纪录片/动画/播客/直播/短视频）
- 单集时长策略（短视频 ≤60s / 中视频 5-15min / 长视频 20-60min / 播客 30-90min）
- 系列规划（单集独立 vs 连续剧 vs 模块化系列）

### D-AV2. 脚本与叙事结构
- 叙事弧线设计模板（开场钩子→主体展开→高潮/转折→总结→CTA）
- 分镜脚本标准模板：场景号 / 画面描述 / 对白/旁白 / 字幕 / 预估时长 / 备注
- 前 5 秒/30 秒的钩子设计（信息差/好奇心/痛点共鸣/数据冲击 四种策略）
- CTA（行动号召）设计（订阅/点赞/评论/链接点击/购买 按优先级排列）

### D-AV3. 制作技术方案
- 设备选型建议（摄像机/麦克风/灯光/提词器/采集卡 按预算分级）
- 录制环境要求（声学处理方案/布光三灯方案/绿幕/背景设计）
- 技术规格定义：分辨率(1080p/4K)、帧率(24/30/60fps)、码率、音频采样率(48kHz)、色彩空间
- 直播推流方案（OBS/vMix/StreamYard + RTMP 推流地址 + 备用流）

### D-AV4. 后期制作流程
- 剪辑风格与节奏指南（快节奏/慢节奏/混合 及转场风格）
- 特效/调色 LUT 风格定义
- 字幕制作（SRT/VTT/ASS 格式，多语言需求，字幕样式规范）
- BGM/音效版权管理（版权来源 + 授权方式 + 替代方案）
- AI 辅助工具清单（语音合成 / 自动字幕 / 降噪 / 视频增强 / 背景移除）

### D-AV5. 发布与分发
- 目标平台清单与优先级（B站/YouTube/抖音/TikTok/小红书视频号/播客平台）
- 各平台的格式与时长适配表
- 缩略图/封面设计策略（A/B 测试方案，点击率优化）
- 发布文案与 SEO 优化（标题/标签/描述/合集/话题）
- 多平台分发工具（一键分发 vs 手动适配）

## Step B 差异化：音视频专家评审

1. 媒体制作专家 (MEDIA_PRODUCER) — 制作流程的完整性与可行性
2. SEO专家 (SEO_EXPERT) — 标题/标签/描述/缩略图的搜索与推荐优化
3. 内容审核专家 (CONTENT_REVIEWER) — 内容合规性、版权风险与受众适配
4. 系统架构师 (ARCH) — 直播/分发/存储技术方案的可行性评估

## Step C 差异化：制作执行

1. 详细分镜脚本定稿（含每个镜头的完整描述和时间标注）
2. 拍摄/录制计划表（日期/地点/人员/设备清单/备用方案）
3. 录制执行清单（开拍前检查/录制中监控/收工后备份 SOP）
4. 后期制作任务看板（粗剪→精剪→调色→音效→字幕→导出 甘特排期）
5. 字幕文件生成（SRT/VTT 标准格式，含时间码校对）
6. 缩略图/封面设计稿（多版本供 A/B 测试）

## Step D 差异化：制作质量门禁

1. 内容审查（事实准确性、合规性、品牌一致性、敏感内容筛查）
2. 技术质量检查表：音画同步 / 音量标准(-14 LUFS) / 分辨率达标 / 无跳帧 / 无削波
3. 字幕准确性验证（语音识别校对 + 时间轴验证）
4. 版权审查（BGM/素材/字体/人物肖像权 逐一核对授权）
5. 多平台预览验证（在各目标平台的实际渲染效果检查）
6. 无障碍检查（字幕/SDH/音频描述/色彩对比度）

## Step E 差异化：音视频交付物

1. 成片/成品文件（母版 + 各平台适配版本）
2. 分镜脚本终稿（含修订记录）
3. 字幕文件包（SRT/VTT/ASS，含多语言版本如适用）
4. 缩略图/封面资源包（含源文件和多平台尺寸变体）
5. 发布文案与 SEO 元数据（各平台定制版本）
6. 项目文件归档（工程文件/素材清单/授权文件）
7. 制作总结与复盘报告（含数据分析和改进建议）
